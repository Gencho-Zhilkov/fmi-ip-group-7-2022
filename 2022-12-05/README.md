### Въпроси:

1. Каква е разликата между `a` и `b`?

```cpp
int *a = new int(7);
int *b = new int[7];
```

2. Какъв е резултата от следните фрагменти код?
```cpp
void foo(int &x) {
	x *= x;
}

int main() {
	int c = 5;
	foo(c);
	std::cout << c << std::endl;
	return 0;
}
```

3. Какво прави следната функция?

```cpp

	if (!a)
		return;
	while(*b++ = *a++);
}
```
